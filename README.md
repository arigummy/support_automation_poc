# Пет проект для Everylab.code

## Суть проекта: хаб для мониторинга и поддержки

## Краткое описание

Веб-приложение для для обработки мониторинга транзакций с возможностью реагирования на инцинденты, и формированием автоматических отчётов.
Проект разбит на четыре подпроекта, реализующих одну из 4-х функциональных единиц хаба:

- Обработка транзакций
- Обнаружение аномалий
- База знаний
- Формирование отчтов

## Сборка проекта

Что-нибудь натыкать через *make*
